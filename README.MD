# Portal Acadêmico DSM - FATEC Jacareí

Este projeto é uma aplicação web full-stack desenvolvida para o curso de **Desenvolvimento de Software Multiplataforma (DSM)**. O sistema consiste em uma interface institucional informativa e um módulo de autenticação de usuários.

---

## Funcionalidades

* Página Inicial (Landing Page): **Informações detalhadas sobre a grade curricular e objetivos do curso de DSM.**
* Sistema de Autenticação:
    * Tela de **Login** com validação de credenciais.
    * Tela de **Cadastro** de novos usuários com confirmação de senha.
* Navegação Dinâmica:
    * Rotas integradas entre o front-end e o servidor back-end.

---

## Tecnologias Utilizadas

 * Front-end: **HTML5, CSS3** (Design responsivo e clean).
 * Back-end: **Node.js** (Servidor rodando na porta 3001).
  * Gerenciamento de Versão: **Git & GitHub.**
  * Hospedagem: **Render** (Web Service).
     Acesse o projeto online: https://site-dsm-fatec.onrender.com/
